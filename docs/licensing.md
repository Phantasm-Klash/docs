# Licensing Policy

## Default Licenses

- Code: MIT License.
- Documentation and website text: CC BY 4.0.
- Project metadata: MIT when it is part of build tooling; CC BY 4.0 when it is prose.

## Asset Rules

No production media asset should enter a repository without a license record.

Allowed asset sources include:

- CC0 or public domain assets;
- MIT, Apache-2.0, BSD, or similarly permissive assets;
- CC BY assets with attribution records;
- original commissioned assets with written permission;
- assets created directly for this project.

Avoid or review carefully:

- copyleft media where the scope is unclear;
- free asset packs with non-commercial limits;
- AI-generated assets whose source terms are unknown;
- fan works or derivative media without explicit redistribution rights.

## Asset Manifest Fields

Every imported asset should record:

- file path;
- source URL or origin;
- author or organization;
- license;
- modification status;
- modification notes;
- import date.
