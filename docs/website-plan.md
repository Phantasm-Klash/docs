# Promotional Website Plan

The promotional website starts as a static site in `website/`.

## First Public Version

The first version should include:

- project name and one-line pitch;
- component links for Gensoulkyo and SpellKard;
- current development status;
- license summary;
- contribution entry points;
- placeholder media until licensed screenshots or concept art exist.

## Later Versions

Later versions can add:

- playable prototype download links;
- match screenshots;
- card gallery;
- development posts;
- server self-hosting guide;
- press kit.
