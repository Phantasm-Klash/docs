# Website

This folder contains the promotional website for Phantasm Klash.

The current version is intentionally static and asset-light. Replace placeholders only with media that has a recorded license.

## GitHub Pages

The site is deployed from this folder by `.github/workflows/deploy-pages.yml` in the docs repository.

For the Phantasm-Klash organization homepage, mirror this folder into the `Phantasm-Klash.github.io` repository and set GitHub Pages to deploy from GitHub Actions in that repository.
