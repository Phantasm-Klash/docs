# Contributing

Thanks for helping build Phantasm Klash.

Early work should focus on:

- clarifying server-client contracts;
- building the first playable SpellKard prototype;
- defining Gensoulkyo persistence and replay data;
- improving documentation and licensing records.

Before opening a pull request, check that:

- the change belongs in this repository rather than a component repository;
- no secrets or closed platform files are included;
- new media assets have license records;
- docs are updated when behavior or scope changes.
