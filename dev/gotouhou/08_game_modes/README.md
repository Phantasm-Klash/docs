# 08 Game Modes

本目录定义服务端支持的多模式玩法。所有模式都使用服务器权威结算，客户端只提交输入、卡牌选择、让渡请求或 UI 操作。

## 模式列表

- `certification_ranked`：考证评级模式。
- `battle_royale`：5-10 人大逃杀模式。
- `world_boss`：全服共享血量世界 Boss。
- `instance_boss`：副本 Boss。

## 共同原则

- 每个模式有独立 `mode_ruleset_version`。
- 每个模式可有独立卡池、卡牌效果、奖励和排行榜。
- 多人模式中的随机、候选卡生成、Boss 行为、回合顺序都由服务端生成和记录。
- Replay 保存模式 ID、规则版本、seed、玩家快照、卡池快照、事件流和最终结果。
- i18n 文本只显示模式名称和说明，不参与规则判断。

## 服务端实现边界

开源服务器可以实现所有模式的基础规则。Steam 商业服可在这些模式上叠加官方活动、成就、Steam Inventory 掉落和世界通告，但市场物品发放仍走闭源适配层。
