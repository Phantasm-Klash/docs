# 管理与运营

## 管理能力

- 配置热更新。
- 禁卡和限卡。
- 活动开启和关闭。
- 宝箱池调整。
- 奖励补偿。
- 玩家封禁。
- 异常对局查询。

## 权限

- 管理入口只允许内网或 VPN。
- 管理操作需要账号权限。
- 高风险操作需要二次确认。
- 所有管理操作写审计日志。

## 热更新

可热更新：

- 卡池。
- 活动。
- 禁卡列表。
- 奖励倍率。
- 公告。

不可影响已开始对局：

- ruleset_version。
- 当前 match seed。
- 已锁定 deck_snapshot。

## 补偿

补偿记录：

- compensation_id。
- reason。
- target_users。
- items。
- expires_at。
- claimed_at。

## 审计

审计重点：

- 异常开箱。
- 重复奖励。
- 高胜率异常账号。
- 输入异常。
- 管理员操作。
