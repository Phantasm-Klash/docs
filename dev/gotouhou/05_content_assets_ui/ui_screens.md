# UI 界面

## 登录

- 自建账号登录。
- 匿名登录仅用于开发和开源私服。
- Steam 商业版显示 Steam 登录状态和购买校验错误。
- 服务器状态。
- 版本检查。

## 主菜单

- 开始匹配。
- 模式选择。
- 卡组。
- 宝箱。
- 创意工坊。
- 活动。
- Replay。
- 设置。

## 匹配

- 当前卡组。
- 模式选择。
- 考证评级、多人混战、世界 Boss、副本 Boss 入口。
- 预计等待。
- 网络状态。
- 取消匹配。

## 卡组

- 卡牌列表。
- 稀有度和类型筛选。
- 卡组费用/类型统计。
- 非法原因提示。
- 保存和设为当前。

## 宝箱

- 宝箱池列表。
- 概率说明。
- 保底进度。
- 单开/十连。
- 结果展示。

Steam 商业版中，可交易物品、市场状态和 Steam 库存同步由闭源适配层展示。开源版只展示本地库存和非市场化宝箱。

## 创意工坊

- 已订阅主题。
- 主题启用/禁用。
- 主题授权说明。
- 缺失资源回退到基础主题。
- 东方主题和其他高风险主题需要明确来源与授权提示。

## 对局 HUD

- 分数。
- 倍率。
- 擦弹数。
- Bomb。
- 能量。
- 手牌。
- 网络状态。
- 对局时间。
- 模式状态。

所有 HUD 文本都来自 i18n key。东方主题可以把“能量”显示为“符力”，把“卡牌”显示为“符卡”。

## 模式界面

- 考证评级：当前评级、rank 分、服务器前 30% 进度、下一阶段资格。
- 大逃杀：公共卡池、当前回合、选择倒计时、3 选 1 候选卡、第零回合顺序。
- 世界 Boss：每日次数、Boss 总血量、当前队伍伤害、让渡卡牌、世界通告。
- 副本 Boss：Boss 血量、阶段机制、通关条件、星级条件、队伍状态。

## 结算

- 胜负。
- 分数明细。
- 奖励。
- 宝箱掉落。
- 任务进度。
- 保存 Replay。
- 再来一局。

## Replay 列表

- 时间。
- 对手。
- 模式。
- 结果。
- 分数。
- 版本。
- 播放、收藏、删除。
