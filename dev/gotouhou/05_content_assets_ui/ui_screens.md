# UI 界面

## 登录

- 自建账号登录。
- 匿名登录仅用于开发和开源私服。
- Steam 商业版显示 Steam 登录状态和购买校验错误。
- 服务器状态。
- 版本检查。

## 主菜单

- 首页只保留四个一级入口：游玩、收藏、社区、玩家设置。
- 游玩入口：练习、匹配、房间、考证评级、大逃杀、世界 Boss、副本 Boss。
- 收藏入口：卡组、宝箱、Replay、创意工坊主题。
- 社区入口：活动通告、好友、社交媒体、推广链接、创意工坊社区内容。
- 玩家设置：输入、手柄移动速度曲线、按键修改、音量、画面和分辨率。
- 高级/开发设置：弹幕实验室、平衡模拟、延迟测试、可访问性细项，放在玩家设置或对应二级页中，不作为首页入口。

当前 SpellKard 已实现 `ClientShellModel` + `UIScreenModel` 的菜单壳：首页四入口、游玩入口、收藏入口、考证入口、社区入口、好友、推广链接、玩家设置、输入设置、音频设置、画面设置、以及高级调试设置均可通过键盘行导航打开和执行。首页隐藏 gameplay/HUD 和旧 dashboard 卡片，只展示大 portrait/standee 区、紧凑状态和 Play/Collection/Community/Player Settings 四个主按钮；考证、匹配、卡组、宝箱、Replay、活动、好友、社交媒体、推广链接和设置细节都下沉到二级/三级页面。每个 UI row 已带有 `section` 元数据，用于区分游玩、收藏、社区、设置、输入、手柄、按键、音频、音量、画面、可访问性、通告、好友、链接、匹配、业务网络和战斗网络等分组；同时带有 `ui_control` 元数据，用于区分页面入口、状态行、按钮、开关、滑条、选择器、链接、好友动作、队列动作、模式动作、网络动作、宝箱/卡牌/Replay 动作和活动领取。设置项还带有可渲染控件值：选择器提供 `control_options`/`control_option_index`，滑条提供 `control_value`/`control_min`/`control_max`/`control_step`/`control_unit`，开关提供布尔 `control_value`。临时 Godot `Control` overlay 已支持长菜单滚动显示选中行、显示本地化导航路径、分区摘要条、分区标题、控件预览、选中项分区/控件类型/控件状态、详情与 Enter 操作提示；正式美术、布局、动画和鼠标/手柄焦点导航仍未完成。

## 匹配

- 当前卡组。
- 模式选择。
- 考证评级、多人混战、世界 Boss、副本 Boss 入口。
- 预计等待。
- 网络状态。
- 取消匹配。

## 玩家设置

- 输入配置：左右手键盘预设、按键修改、恢复当前预设。
- 手柄：死区、灵敏度、移动速度曲线、震动强度。
- 音频：Master、Music、SFX、UI、Voice 分组音量；关键事件视觉提示；高频擦弹音抑制。
- 画面：分辨率、窗口/无边框/全屏、VSync、FPS 上限、屏幕震动、背景压暗。
- 可访问性：低闪烁、简洁背景、判定点常显、练习擦弹圈、弹幕透明度、色弱友好配色。

当前实现中，手柄左摇杆曲线已接入本地移动输入；音量修改会同步到 Godot `AudioServer` bus；分辨率、窗口模式、VSync 和 FPS 上限会同步到运行时显示设置。输入、手柄、按键、音频、音量、画面和可访问性 row 已被分区，并且按键/曲线/分辨率类 row 标记为选择器且带候选项，音量/灵敏度/死区/震动/震屏类 row 标记为滑条且带范围，VSync 和可访问性开关标记为开关且带布尔状态。后续正式 Godot Control 场景应复用这些元数据作为设置页左侧导航、页内分组和控件渲染依据。

## 卡组

- 卡牌列表。
- 稀有度和类型筛选。
- 卡组费用/类型统计。
- 非法原因提示。
- 保存和设为当前。

## 宝箱

- 宝箱池列表。
- 概率说明。
- 保底进度。
- 单开/十连。
- 结果展示。

Steam 商业版中，可交易物品、市场状态和 Steam 库存同步由闭源适配层展示。开源版只展示本地库存和非市场化宝箱。

## 创意工坊

- 已订阅主题。
- 主题启用/禁用。
- 主题授权说明。
- 缺失资源回退到基础主题。
- 东方主题和其他高风险主题需要明确来源与授权提示。

## 对局 HUD

- 分数。
- 倍率。
- 擦弹数。
- Bomb。
- 能量。
- 手牌。
- 网络状态。
- 对局时间。
- 模式状态。

所有 HUD 文本都来自 i18n key。东方主题可以把“能量”显示为“符力”，把“卡牌”显示为“符卡”。

## 模式界面

- 考证评级：当前评级、rank 分、服务器前 30% 进度、下一阶段资格。
- 大逃杀：公共卡池、当前回合、选择倒计时、3 选 1 候选卡、第零回合顺序。
- 世界 Boss：每日次数、Boss 总血量、当前队伍伤害、让渡卡牌、世界通告。
- 副本 Boss：Boss 血量、阶段机制、通关条件、星级条件、队伍状态。

## 结算

- 胜负。
- 分数明细。
- 奖励。
- 宝箱掉落。
- 任务进度。
- 保存 Replay。
- 再来一局。
