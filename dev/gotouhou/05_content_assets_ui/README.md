# 05 Content Assets UI

本阶段定义素材、机体、弹幕视觉、界面、音频、i18n 和可访问性。目标是让基础游戏使用原创主题，同时支持多套主题资源。东方相关文本和美术不进入基础包，后续通过 Steam 创意工坊 Mod 或授权主题包替换。

## 目标

- 开源素材管线和多主题素材结构。
- 初版 4 个机体。
- 清晰可读的弹幕视觉。
- 完整 UI 流程。
- 可重绑定输入和低闪烁设置。
- i18n 文本替换和主题包加载。

## 原则

- 可读性优先于华丽特效。
- PVP 干扰不能遮挡关键信息。
- 每个素材必须有授权记录。
- 每个玩家可见文本必须来自 i18n key。
- UI 第一屏进入实际游戏功能，不做纯营销落地页。
