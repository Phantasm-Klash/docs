# 可访问性与输入

## 输入

- 键盘。
- 手柄。
- 可重绑定按键。
- 支持左右手方向键布局。

## 默认键位

- 移动：方向键/WASD。
- 射击：Z。
- Bomb：X。
- 低速：Shift。
- 卡牌：1-4。
- 暂停：Esc。

## 视觉可访问性

- 低闪烁模式。
- 简洁背景。
- 弹幕透明度调整。
- 判定点常显。
- 擦弹圈练习模式显示。
- 色弱友好弹幕配色预设。

## 音频可访问性

- 分组音量。
- 关键事件视觉提示。
- 可关闭高频擦弹音。

## 网络提示

- 显示 ping 和连接质量。
- 排位网络不佳时明确提示。
- 重连时不隐藏对局状态。

## 验收

玩家只使用键盘或手柄都能完整完成登录、匹配、对局、卡组和宝箱流程。Steam 商业版还应能完成创意工坊主题启用和库存查看。
