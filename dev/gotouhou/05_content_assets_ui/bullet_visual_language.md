# 弹幕视觉语言

## 目标

玩家必须能快速判断弹幕危险程度、速度、方向和归属。视觉风格可以华丽，但不能牺牲判定可读性。

## 弹种区分

- 小圆弹：基础低威胁。
- 大玉：高威胁、大判定、低密度。
- 激光：持续威胁，提前预警。
- 星弹：中速装饰弹，判定清晰。
- 诱导弹：带尾迹或方向提示。

## 颜色

- 不同速度和危险等级使用可区分颜色。
- 对方卡牌修改过的弹幕添加边框或粒子提示。
- 不能让弹幕与背景同色。

## 判定可读性

- 子弹贴图半径可以大于判定半径，但不能小于判定半径。
- 高密度阶段降低特效透明叠加。
- 判定点永远处于清晰图层。

## PVP 干扰限制

- 不能遮挡自机判定点。
- 不能遮挡 Bomb、血量、手牌费用等关键 UI。
- 视觉预警干扰必须有上限。

## 背景

背景应低对比、少闪烁、少快速移动。练习和排位模式提供简洁背景选项。
