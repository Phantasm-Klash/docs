# 机体与射击类型

## 初版机体

### 均衡型

- 高速移动中等。
- 低速射击集中。
- Bomb 标准清屏。
- 适合新手。

### 低速精密型

- 低速速度更低。
- 判定点更醒目。
- 擦弹收益更高。
- 高速机动较弱。

### 广范围型

- 高速射击覆盖宽。
- 低速也保留侧向辅助弹。
- 擅长处理分散目标。
- 精密擦弹收益普通。

### 符力型

- 擦弹和上线获得更多符力。
- 依赖卡牌节奏。
- Bomb 稍弱。
- 适合构筑玩家。

## 机体数据

每个机体配置：

- move_speed。
- slow_speed。
- focus_shot_pattern。
- normal_shot_pattern。
- bomb_type。
- graze_modifier。
- spell_power_modifier。

## 平衡目标

机体影响打法，不提供绝对强度碾压。排位中每个机体应有可用卡组和反制空间。
