# 音频与音乐政策

## 授权

音频遵循与图像相同的授权要求。优先使用：

- CC0 音效。
- 可商用 CC-BY 音乐。
- 自制或委托音乐。
- 明确允许游戏使用的素材包。

禁止使用：

- 东方官方音乐。
- 未授权 remix。
- 来源不明的音效包。
基础发行包禁止内置东方官方音乐或未经授权同人音乐。东方音乐替换只允许通过授权主题包或 Workshop Mod 进入。

## 音频分类

- BGM。
- UI 音效。
- 射击音效。
- 擦弹音效。
- 命中音效。
- Bomb 音效。
- 卡牌发动音效。

## 混音

- 擦弹音效必须轻，不因高频触发刺耳。
- 卡牌音效要有识别度。
- 低血、终幕和网络警告不能互相遮盖。

## 设置

音量分组：

- 主音量。
- 音乐。
- 音效。
- UI。
- 语音预留。

## 验收

所有进入构建的音频都必须在 license manifest 中记录来源和协议。
