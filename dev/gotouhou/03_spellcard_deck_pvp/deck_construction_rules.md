# 卡组构筑规则

## 初版规则

- 卡组数量：20 张。
- 起手：4 张。
- 手牌上限：6 张。
- 同名卡：最多 2 张。
- 高稀有卡：最多 6 张。
- 强干扰卡：最多 4 张。

## 费用系统

- 施放卡牌消耗能量。
- 能量来自存活、擦弹、上线风险和部分卡牌。
- 基础主题显示为“能量”，东方主题可通过 i18n 显示为“符力”。
- 高影响卡必须有更高费用或更长冷却。

## 抽牌规则

- 对局开始抽起手。
- 固定 tick 或固定时间抽牌。
- 手牌满时新牌进入弃牌或延迟抽取，初版建议延迟抽取。

## 模式限制

- 练习模式可使用全部已拥有卡。
- 休闲模式允许活动卡。
- 排位只允许当前 ruleset_version 的平衡卡池。

## 服务端校验

进入匹配前服务端校验：

- 卡牌拥有权。
- 同名数量。
- 稀有度限制。
- 禁卡列表。
- ruleset_version。

对局开始时保存 deck_snapshot，后续库存变化不影响当前对局。

## 文本与主题

卡组、卡牌名称和卡牌描述只保存 code 与 i18n key。基础游戏显示“卡牌”，东方主题显示“符卡”只能通过主题文本包完成，不能改变卡组规则。
