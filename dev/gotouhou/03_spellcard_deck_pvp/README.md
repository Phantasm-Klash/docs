# 03 Card Deck PVP

本阶段定义卡牌、卡组和 PVP 平衡。卡牌既是局内战术资源，也是局外收集和构筑目标。目录名保留 `spellcard_deck_pvp` 是早期命名；基础游戏内文本必须显示为“卡牌”，东方主题包可通过 i18n 显示为“符卡”。

## 初版目标

- 20 张卡组。
- 起手 4 张。
- 手牌上限 6。
- 同名最多 2 张。
- 强干扰卡数量限制。
- 服务器校验全部卡组和施放条件。

## 设计原则

- 卡牌可以增益自身、修改对方弹幕或修改双方环境。
- 所有效果都必须可反制、可读、可记录。
- 核心玩法卡必须可通过免费路径获得。
- 宝箱和掉落扩展构筑深度，但不能制造不可反制的数值碾压。
