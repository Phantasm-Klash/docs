# 自动化测试

## 服务端单元测试

覆盖：

- 卡组校验。
- 卡牌费用和冷却。
- 卡牌效果叠加。
- 开箱概率。
- 保底。
- 重复卡转粉尘。
- 奖励幂等。
- 非法请求拒绝。

## 对局模拟测试

覆盖：

- 固定输入流产生固定结果。
- 同一 seed 多次运行结果一致。
- Bomb 和决死窗口。
- 擦弹只计一次。
- 无敌期间收益降低。

## 数据库测试

覆盖：

- 迁移可重复执行。
- 必要索引存在。
- 经济流水与钱包一致。
- match_id 奖励只能结算一次。

## 客户端测试

覆盖：

- 输入映射。
- 低速切换。
- Replay 播放。
- UI 非法卡组提示。
- i18n key 缺失检测。
- 主题包 manifest 校验。
- 考证评级授予、rank 分和 top 30% 资格。
- 大逃杀公共卡池、3 选 1 和第零回合顺序。
- 世界 Boss 每日次数、让渡卡牌、持久血量扣除和世界通告。
- 副本 Boss 通关条件、失败结算和星级。

## Steam 闭源测试

闭源 CI 额外覆盖：

- Steam 登录 ticket 校验。
- App 所有权校验。
- 成就解锁。
- Workshop 订阅和资源加载。
- Steam Inventory 掉落和开箱。
- 市场可交易属性校验。

## CI

每次提交至少运行服务端单元测试、配置校验和文档链接检查。
