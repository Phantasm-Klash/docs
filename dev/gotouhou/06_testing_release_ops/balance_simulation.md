# 平衡模拟

## 目标

通过批量模拟和真实对局数据发现过强卡牌、无效卡牌、异常组合和机体差异。

## 输入

- 卡牌配置。
- 卡组列表。
- 机体配置。
- 弹幕规则。
- 模式规则。
- 玩家 skill profile。
- 随机 seed 列表。

## 输出

- 胜率。
- 平均分。
- 平均擦弹。
- 平均受击。
- 平均出牌次数。
- 单卡收益。
- 组合收益。
- 模式专用收益，如大逃杀排名、Boss 秒伤、评级晋级率。

## 模拟层级

- 规则级：不渲染，只跑服务端模拟。
- 卡组级：批量卡组对战。
- 真人数据级：导入真实 match_events 做统计。

## 告警

- 胜率异常。
- 出场率异常。
- 对局时间异常。
- 某卡造成性能异常。
- 某组合反制率过低。
- 某评级晋级率异常。
- 大逃杀第零回合顺序带来过高先手优势。
- Boss 让渡卡牌造成输出爆炸组合。

## 使用节奏

每周平衡前运行一次完整模拟。新卡上线前必须通过模拟和小规模测试。
